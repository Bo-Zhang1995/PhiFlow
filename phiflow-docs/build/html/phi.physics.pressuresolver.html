solve(divergence, domain, pressure_guess)¶

Solves the pressure equation Δp = ∇·v for all active fluid cells where active cells are given by the active_mask. The resulting pressure is expected to fulfill (Δp-∇·v) ≤ accuracy for every active cell.

Parameters

  • divergence – the scalar divergence of the velocity channel, ∇·v

  • domain – DomainState object specifying boundary conditions and active/fluid masks. The domain must be equal for all examples (batch dimension equal to 1).

  • pressure_guess – (Optional) Pressure channel which can be used as an initial state for the solver

Returns

pressure tensor (same shape as divergence tensor), number of iterations (integer, 1D integer tensor or None if unknown)

Builds a sparse matrix such that when applied to a flattened pressure channel, it calculates the laplace of that channel, taking into account obstacles and empty cells.

Parameters

  • dimensions – valid simulation dimensions. Pressure channel should be of shape (batch size, dimensions…, 1)

  • extended_active_mask – Binary tensor with 2 more entries in every dimension than ‘dimensions’.

  • extended_fluid_mask – Binary tensor with 2 more entries in every dimension than ‘dimensions’.

Returns

SciPy sparse matrix that acts as a laplace on a flattened pressure channel given obstacles and empty cells
